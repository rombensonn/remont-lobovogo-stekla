# Лендинг «Ремонт и тонировка автостёкол»

Готовый одностраничный сайт для узкой мастерской по автостёклам: ремонт сколов и трещин, тонировка, растонировка и бронь лобового стекла. Фронтенд сделан на Next.js 15, TypeScript, Tailwind CSS и Framer Motion. Заявки принимает отдельный PHP endpoint `/api/lead.php`.

## Быстрый запуск

```bash
npm install
npm run dev
```

Фронтенд откроется на `http://localhost:3000`.

Для локальной проверки PHP endpoint:

```bash
php -S localhost:8080
```

Чтобы форма из Next.js dev-сервера отправлялась в PHP:

```bash
NEXT_PUBLIC_LEAD_ENDPOINT=http://localhost:8080/api/lead.php npm run dev
```

## Проверка перед публикацией

```bash
npm run lint
npm run build
```

Проверка PHP через `curl`:

```bash
curl -X POST http://localhost:8080/api/lead.php \
  -F "name=Тест" \
  -F "phone=+79990000000" \
  -F "service=ремонт скола" \
  -F "personalDataConsent=1" \
  -F "privacyPolicyConsent=1"
```

Ответ должен быть JSON с `success: true`. Резервная копия заявки сохраняется в `storage/leads.csv`.

## Деплой на обычный PHP-хостинг

1. Замените все плейсхолдеры в коде и `api/config.php`.
2. Выполните `npm install && npm run build`.
3. Загрузите содержимое папки `out/` в корень сайта на хостинге.
4. Рядом с файлами статического сайта загрузите папку `api/`.
5. Создайте папку `storage/` в корне сайта и дайте PHP право записи.
6. Убедитесь, что `storage/.htaccess` запрещает прямой доступ к сохранённым заявкам и фото.
7. Проверьте отправку формы. Если `mail()` на хостинге не работает, замените отправку на SMTP через PHPMailer.

## Что заменить перед запуском

- `[НАЗВАНИЕ]`
- `[EMAIL]`
- `[ИП/ООО]`
- `[ИНН]`
- `[ОГРН/ОГРНИП]`

Адрес, телефон, город, район и ссылка на Яндекс Карты уже подставлены из текущих данных.

Также замените `https://example.com` в `lib/site-data.ts`, `.env.example` и `api/config.php` на реальный домен.

## Важные заметки

- В проекте не указаны выдуманные гарантии, бренды плёнок, сертификаты или реальные фото работ.
- Видимые отзывы — короткие переработанные формулировки по предоставленным смыслам, без публикации негативных отзывов.
- Цены указаны как ориентиры. Финальная стоимость зависит от автомобиля, повреждения и материалов.
- Форму нельзя отправить без двух обязательных чекбоксов: согласие на обработку персональных данных и ознакомление с политикой.
